Final Year Project

Local Tour Planner System

This Planner System provides users a convenient way to plan their trips by itemising the landmarks to visit and find accommodation and restaurants. Almost everyone loves traveling and they will enjoy it even more if uncertainties are removed. So, this system removes the fears of travelling by making planning convenient and easy, and smoothly integrates the planning and executing stages of travel

Here is the details of the Management Subsystem :

This is the Homepage of the Local Tour Planner System. On the homepage, there is a search destinations panel where user able to search which destination that they want to visit and along with the starting and end date for their trip.

Once user have selected which destination they want to visit, it will redirect them to the destination page. In that page, there will be few plans that shows the nearby places of the searched destination. After user have selected the plan, it will redirect them to this page where it have hotels, restaurant and attraction button will be shown. User shall choose the hotel, restaurant and attraction in the specific page and once user have select either places, the selected places will display on this page. Then, User shall choose the time for each places. The time module for this plan uses an algorithm to calculate the time difference between two points by getting the location distance and time. With this, User able to know the estimate arrival time for each location.

This is the itinerary page of Local Tour Planner System. This page shows the selected plan which consists of hotels, destination and food which the user have chosen for their trip. The selected time and date will be displayed. User also able to generate or download a PDF file for their itinerary according to their plan details.
